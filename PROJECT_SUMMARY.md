## 🚀 部署方案

### 快速部署（3步完成）

1. **Fork仓库**
   ```bash
   # 在GitHub网页端点击Fork按钮
   ```

2. **启用Pages**
   ```
   Settings → Pages → Source: Deploy from branch → main
   ```

3. **访问网站**
   ```
   https://yourusername.github.io/news-hub
   ```

### 自动化流程

- **触发条件**: Push到main分支 + 每小时定时
- **构建过程**: 更新数据 → 生成静态文件 → 部署到Pages
- **更新频率**: 每小时自动更新最新热点
